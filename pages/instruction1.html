Instruktion 1: Variabler och Grundläggande Beräkningar

Steg för steg guide:

  1. Öppna din kodeditor och skapa en ny fil som heter script.js
  2. Deklarera två variabler med const: 
    const tal1 = 10;
const tal2 = 5;
  3. Skapa en funktion för addition: 
    function addera(a, b) {
    return a + b;
}
  4. Anropa funktionen och spara resultatet: 
    const summa = addera(tal1, tal2);
  5. Skriv ut resultatet i konsolen: 
    console.log('Summan är: ' + summa);

Instruktion 2: Stränghantering

Steg för steg guide:

  1. Deklarera en variabel med let för ett namn: 
    let namn = 'Anna';
  2. Skapa en funktion som skapar en hälsning: 
    function skapaHälsning(namn) {
    return `Hej ${namn}, välkommen!`;
}
  3. Testa funktionen med olika namn: 
    console.log(skapaHälsning(namn));
namn = 'Erik';
console.log(skapaHälsning(namn));
