CSS Animation

Instruktion 1: Hover Effekt med Transition

Steg för steg guide:

  1. Skapa en HTML-knapp: 
    <button class="hover-button">Hover mig!</button>
  2. Lägg till grundläggande CSS för knappen: 
    .hover-button {
    padding: 10px 20px;
    background-color: #3498db;
    color: white;
    border: none;
    border-radius: 5px;
    transition: all 0.3s ease;
}
  3. Lägg till hover-effekten: 
    .hover-button:hover {
    background-color: #2980b9;
    transform: scale(1.1);
}

Instruktion 2: Rörlig Animation med Keyframes

Steg för steg guide:

  1. Skapa ett div-element: 
    <div class="moving-box"></div>
  2. Definiera keyframes-animationen: 
    @keyframes moveRight {
    from {
        transform: translateX(0);
    }
    to {
        transform: translateX(200px);
    }
}
  3. Styla div-elementet och lägg till animationen: 
    .moving-box {
    width: 50px;
    height: 50px;
    background-color: #e74c3c;
    animation: moveRight 2s ease infinite alternate;
}
